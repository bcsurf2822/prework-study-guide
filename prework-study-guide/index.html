HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The child elements are nested inside other elements
  • The parent element is the element that holds all the children
  • Elements and tags differ in that elements is the entire element from opening to closing
  • Tags is only what appears inside of brackets
  • Semantic HTML elements describes content and role it plays on a webpage and makes HTML easy to read to other developers
  • HTML is the content users see
  • Elements are made up of text image headers and list
  • Elements describe content to the browser and other developers
  • Declarations are not case sensitive
  • HTML elements are case sensitive
  • HTML attribute defines additional info about an element they are always defined in beggining and not at the end

CSS

  • CSS Cascading Style Sheets
  • CSS has to do with the syle on the webpage
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • content is presented through color aesthetics layout organization
  • A wildcard aka asterisk makes all element on page the same
  • You can combine 2 selectors into a single rule with a comma
  • DRY code do not repeat yourself code is encouraged to reduce the number of lines of code, reduce size and complexity
  • Border assigns a border to page

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • A Repository is a folder where all files and version hisories are stored
  • A branch is a workspace to make changes that will not affect the live switches
  • Markdown turn text into GitHub code
  • A pull Resquest is to ask for a change in one branch to be merged into another, usually after a review
  • A merge is when the pr is approved, it is merged onto main branch and deployed on a live site
  • We use issues to track our work. It allows users to report new tasks and content fixes and allows users to track their progress on a project board
  • Federalist is a platform that securely deploys a website from GitHub repo and lets users preview proposed and published changes
  • It is important to memorize and practice Git flow

JavaScript

  • JavaScript adds function to your website similar to the way that you would use your house
  • javascript is an extremely useful tool that allows a user to interact with site
  • You can add comments to your code by useing /* followed by */
  • The browser will ignore anything between those symbols in lines of code
  • Conditionals are code structures used to test if an expression returns true or not and example is the if else statement
  • JavaScript is case sensitive
  • We need variables to do anything intersting in programming
  • You can use JavaScript to create games, incorporate 2d and 3d graphics and create comprehensive database driven apps
  • JavaScript is compact but extremely flexible
  • The inspector mode is an extremely useful tool in coding. You can see how a website is structured and can be useful in debugging
